# 美本申请第二大脑 (US College App Second Brain)

> **第一性原理：招生官在找活生生的人。**
> 本项目是一个基于各大Vibe Coding Agent的深度交互框架，旨在通过三层记忆架构与时间轴感知（M值），帮助申请者挖掘真实自我，拒绝申请同质化。

## 🛠 v2.2 更新说明
- **时间轴感知 (M-Value)**：根据申请季进度动态调整 AI 建议的权重。
- **架构重组**：合并了 v2.1 的逻辑稳定性补丁，强化了本体画像的深度。
- **排版优化**：统一了 Markdown 交互规范，提升了长上下文下的指令遵循度。

## 🚀 快速启动
1. 将 `CLAUDE.md` 的内容导入你的 AI 对话环境。
2. 确保你的文件目录符合 `templates/` 中的结构。
3. 对 AI 说“启动”。
