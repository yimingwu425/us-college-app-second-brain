# 7. 启动、初始化与记忆读取

## 触发条件

- 用户说"启动"时执行。
- Terminal 用户自动执行。
- 第一次会话：`本体画像/00-核心身份.md` 为空或不存在。
- 非第一次会话：核心身份文件已经存在且有内容。

## 需要读取的文件

**第一次会话**：

- 不读其他文件，直接进入对话四阶段的"阶段一"。

**非第一次会话**：

1. 读取 `本体画像/00-核心身份.md`（必读）。
2. 读取 `本体画像/01-价值观与原则.md`（必读）。
3. 读取核心身份.md 中的"学制"和"申请季开始时间"，计算 M 值，得出当前阶段。
4. 读取 `申请追踪/Deadline总览.md`（检查近期待办）。
5. 读取 `记忆库/语义记忆/申请故事主线.md`（了解当前人设版本）。

**按需读取（不要主动加载）**：

- 其他本体画像模块：仅在相关任务时读取。
- 记忆库的情景记忆：仅在用户询问历史时读取。
- 日记、素材库：仅在用户明确要求时读取。
- 错题本、学校研究：仅在对应话题触发时读取。

## 操作步骤

**第一次会话**：

1. 自动创建初始化文件。
2. 直接进入对话四阶段的"阶段一"。
3. 输出阶段一的固定开场白。

**非第一次会话**：

1. 根据核心身份.md 中的"学制"和"申请季开始时间"计算 M 值。
2. 结合 `申请追踪/Deadline总览.md` 判断是否有近期待办。
3. 结合 `记忆库/语义记忆/申请故事主线.md` 了解当前人设版本。
4. 按下面"晨间简报模板"输出开场白。

### 晨间简报模板

**有待办时**：

```markdown
✓ 今天是 YYYY-MM-DD，距离申请季还有 M 个月（当前：XX 期）

有几件事要注意：
- [事项] | 截止 MM/DD

今天想跟我分享什么？
```

**没有待办时**：

```markdown
✓ 今天想跟我分享什么？
```

## 风险检查

- 第一次会话不要读取其他记忆文件，避免用空档案假装了解用户。
- 非第一次会话只读启动所需文件，其他内容按需懒加载。
- 用户没填"申请季开始时间"时，可以按学制和年级推断，但必须告知："我是按 XX 推断的，如果不对请更正"。
- 晨间简报不要暴露"模板"、"阶段"等系统结构。
