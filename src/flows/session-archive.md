# 12. 会话归档与记忆更新

## 触发条件

- 每次会话开始时，追加本次会话入口记录。
- 每次对话结束后，自动执行归档与记忆更新。
- 用户说了一件新的经历时，触发情景记忆和语义记忆提炼。
- 发现重复出现的模式（≥2 次）时，触发语义记忆更新。
- 用户表达"明天"、"下周"、"X 月 X 日"、"要交"、"截止"、"deadline"、"ED"、"EA"、"RD"、"计划"、"打算"、"记得"等时间意图时，触发申请追踪更新。

## 需要读取的文件

- `对话历史.md`：只能追加，用于记录会话入口和产出摘要。
- `记忆库/情景记忆/YYYY-MM/`：读取相关近期情景，避免重复记录。
- `记忆库/语义记忆/核心特质.md`：更新用户稳定特质。
- `记忆库/语义记忆/反复出现的主题.md`：更新重复主题和出现次数。
- `记忆库/语义记忆/申请故事主线.md`：更新申请故事主线。
- `记忆库/强制规则/`：记录用户明确设定的偏好和约束。
- `申请追踪/Deadline总览.md`：记录时间意图和截止事项。

## 操作步骤

### 三层记忆架构

```text
记忆库/
├── 情景记忆/        # 每次对话的具体内容
│   └── YYYY-MM/
├── 语义记忆/        # 关于用户的规律和特质
│   ├── 核心特质.md
│   ├── 反复出现的主题.md
│   └── 申请故事主线.md
└── 强制规则/        # 用户设定的偏好和约束
```

**记忆流转机制**：

```text
对话内容
  → 情景记忆（记录发生了什么）
  → 语义记忆（提炼用户是什么样的人）
  → 强制规则（用户说不希望 AI 做的事）
```

**记忆提炼时机**：

- 每次对话结束后。
- 用户说了一件新的经历。
- 发现重复出现的模式（≥2 次）。

**会话结束时自动执行**：

1. 把本次对话关键内容存入情景记忆。
2. 更新语义记忆（用户又透露了什么新的特质？）。
3. 生成下次对话的开场白："上次我们聊到了 XX，你后来有没有想到什么？"

### 时间意图自动捕获

触发后自动执行：

1. 将事项添加到 `申请追踪/Deadline总览.md`。
2. 告知用户："已添加到申请追踪，届时会提醒你"。

### 智能存储路由

每次创建文件时自动执行：

1. 识别内容类型。
2. 查询存储规则。
3. 构建完整路径。
4. 告知用户保存位置。

**存储规则**：

| 内容类型 | 存储位置                   |
| -------- | -------------------------- |
| 对话内容 | `记忆库/情景记忆/YYYY-MM/` |
| 用户特质 | `记忆库/语义记忆/`         |
| 学校笔记 | `学校研究/[学校名]/`       |
| 活动记录 | `素材库/经历/`             |
| 读书感受 | `素材库/感受/读过的书/`    |
| 日常想法 | `素材库/思考/每日想法/`    |
| 日记     | `日记/YYYY-MM/`            |
| 申请进度 | `申请追踪/`                |

### 八步自我迭代流程

完成重大对话后执行：

1. **观察**：这次对话发生了什么？
2. **分析**：用户透露了什么新的特质？
3. **设计**：需要更新哪些文件？
4. **实施**：更新语义记忆、申请故事主线。
5. **验证**：更新的内容符合第一性原理吗？（过第 0 节三个问题）
6. **记录**：存入情景记忆。
7. **提炼**：更新语义记忆。
8. **完成**：更新对话历史.md，记录本次迭代主题。

## 写入或更新

每次会话开始时，在 `对话历史.md` 追加：

```markdown
---
## 会话：YYYY-MM-DD HH:MM

### 用户
[第一条消息摘要]

### AI
[响应摘要]

### 产出
[这次对话提炼出了什么]
```

会话结束、出现新经历或发现重复模式时，写入或更新：

- `记忆库/情景记忆/YYYY-MM/`
- `记忆库/语义记忆/核心特质.md`
- `记忆库/语义记忆/反复出现的主题.md`
- `记忆库/语义记忆/申请故事主线.md`
- `记忆库/强制规则/`
- `申请追踪/Deadline总览.md`

## 风险检查

- `对话历史.md` 禁止删除或修改历史记录，只能追加。
- 记忆更新不能把一次性情绪误判成稳定特质。
- 强制规则只记录用户明确设定的偏好和约束，不要自行发明。
- 时间意图捕获后必须告诉用户已加入申请追踪。
- 创建文件必须按智能存储路由放到正确目录，并告知用户保存位置。
